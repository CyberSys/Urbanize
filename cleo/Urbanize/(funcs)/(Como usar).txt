### O que é:
 Estes arquivos .sc contém funções úteis para serem usadas nos seus scripts. Quando for usar algum, adicione-o usando REQUIRE, de preferência no topo do seu script.

### Lembrando:
 REQUIRE do GTA3script só encontra arquivos .sc a partir de uma pasta com o mesmo nome do script, por exemplo, para carregar o "exemplo.sc" para o "script.sc" é necessário que o "exemplo.sc" esteja dentro de uma pasta com nome "script".

### Funções atualizadas:
 A pasta "cleo\Urbanize\(funcs)" guarda todos os arquivos .sc de funções *ATUALIZADOS*, portanto, sempre use esta pasta para "instalar" as funções nos seus scripts.

### Como usar a pasta (funcs):
 Uma boa dica é simplesmente criar uma pasta com o seu script e copiar a pasta "(funcs)" lá dentro, ou seja, seu Table1.sc terá uma pasta "Table1" e dentro dela terá uma cópia da "(funcs)" com todas as funções.

### Fique atualizado:
 Quando funções da pasta (funcs) atualizarem corrigindo bugs etc, é aconselhável você atualizar a pasta "(funcs)" da pasta do seu script.

### Como funciona as funções:
 Abrindo cada .sc você encontra breves explicações com um exemplo de uso.
 Várias funções estão profundamente explicadas na wiki. Por exemplo: https://github.com/JuniorDjjr/Urbanize/wiki/Obtendo-os-modelos-de-peds-dinamicamente
